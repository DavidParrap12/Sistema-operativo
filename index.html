¿Qué es la gestión de E/S?

La interfaz crítica entre la CPU y el mundo exterior.

input

Concepto de Entrada

Flujo de datos desde dispositivos periféricos (teclado, red, sensores) hacia la memoria del sistema para su procesamiento por la CPU.

output

Concepto de Salida

Envío de datos procesados desde la memoria hacia el exterior (monitores, impresoras, almacenamiento externo o redes).

Funciones principales

tune

Estrategias de Transferencia

sync Síncrona

La CPU espera bloqueada hasta que el dispositivo completa la operación. Fácil de implementar pero ineficiente para multitarea.

sync_alt Asíncrona

La CPU inicia la operación y continúa ejecutando otros procesos. Es notificada mediante interrupciones al terminar.

Técnica Descripción Carga CPU Uso Ideal
Polling Consulta constante del estado del dispositivo. Muy Alta Dispositivos ultra-rápidos.
Interrupciones El dispositivo avisa cuando está listo. Media Teclados, ratones, redes.
DMA Transferencia directa Memoria-Dispositivo. Baja Discos duros, Video 4K.
